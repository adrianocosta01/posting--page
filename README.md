# posting--page
Implementação do projeto de postagem de blog do DevStart

# Projeto de Criação de Postagem

Este projeto é uma página web simples que permite ao usuário criar um
post com título e conteúdo, simulando o funcionamento de redes sociais
como Facebook e LinkedIn.

-[Visite o projeto online](https://adrianocosta01.github.io/posting--page/)

## Funcionalidades

-   Campo para inserir título do post
-   Campo para inserir conteúdo do post
-   Botão para postar
-   Integração com API aberta para simular envio dos dados

## Como usar

1.  Abra o arquivo `index.html` no navegador.
2.  Preencha o título.
3.  Digite o conteúdo do post.
4.  Clique em **Postar**.

## Tecnologias utilizadas

-   HTML
-   CSS
-   JavaScript
-   API pública (simulação)

## Estrutura do projeto

-   `index.html` -- Estrutura da página
-   `style.css` -- Estilos da interface
-   `script.js` -- Lógica de envio e manipulação da interface
